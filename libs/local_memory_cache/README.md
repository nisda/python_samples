# Local Memory Cache

ローカルメモリに値をキャッシュする機能を実現するライブラリ。  

## 機能概要
* キャッシュキーを指定して、ローカルメモリに値を保存する。
* セット時に有効期限の指定が可能。
  有効期限内であれば再セットしても値が更新されず、キャッシュされている値が取得される。
* 有効期限を無視しての強制更新も可能。
* lambda 式を利用して function の実行結果をキャッシュできる。
  キャッシュ有効期間内であれば function は再実行されない。



## 利用形態
以下３種類の利用形態がある。

* CacheContianerCore  
  キャッシュを格納するインスタンスを生成して利用する。
* CacheContainer  
  インスタンス生成が不要。シングルトン。
  共通の領域に保存するキャッシュコンテナ。
* deco_local_mem_cache  
  function に付与するデコレータ。    
  function の実行結果を自動的にキャッシュして再実行を抑止する。  
  パラメータごとに結果を保存する。特定のパラメータのみキャッシュキーにするなどの設定も可能。
