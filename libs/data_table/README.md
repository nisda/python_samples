# DataTable

## 概要



### 対応データ型（INPUT）

| データ型                | データ指向 | カラム名 | 備考 |
| ---                    | -- | -- | -- |
| `Dict[str, List[Any]]` | 列指向  | あり | |
| `List[Dict[str, Any]]` | 行指向  | あり | |
| `Dict[str, Any]`       | 行指向  | あり | 複数列／１行のテーブルとして扱う |
| `List[List[Any]]`      | 行指向  | なし | |
| `List[Any]`            | 行指向  | なし | １列／複数行のテーブルとして扱う。 |
| `Any`                  | 行指向  | なし | それ以外のデータ型。１列／１行のみのテーブルとして扱う。 |
| `None`                 | -      | なし | ０列０行の空テーブルとして扱う |


#### 備考
* Tuple型も使用できる。List型と同じ扱いとする。
* データ型は自動判別される。
  * INPUTデータ全体のデータ型と、1つ目のレコードでデータ型を判別する。  
  * 2つ目以降のレコードで型が異なる場合は処理に任せる。処理できなければ何らかの Exception。
* カラム名は強制的にstr型に変換される。  
* カラム名なしのデータには、カラム名として `0` からの連番[str型]を自動付与する。
* カラム名はinit時、および後からいつでも変更が可能。


### 対応するデータ型（OUTPUT）


| メソッド | データ型                             | データ指向 | カラム名 | 備考 |
| --   | ---                                 | --        | --      | -- |
| rows(type='dict') | `List[Dict[str, Any]]` | 行指向     | あり    | |
| rows(type='list') | `List[List[Any]]`      | 行指向     | なし    | |
| cols(type='dict') | `Dict[str, List[Any]]` | 列指向     | あり    | |
| cols(type='list') | `List[List[Any]]`      | 列指向     | なし    | |
